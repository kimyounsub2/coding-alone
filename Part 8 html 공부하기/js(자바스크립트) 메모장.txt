- 변수 선언하기
var 뒤에 변수 이름 작성
var를 한번만 쓰고 뒤에 여러개의 변수를 선언할수 있음


for문
for(초깃값; 조건; 증가식){
    실행할 명령
}
초깃값 : 카운터 변수를 초기화 한다. 초깃값은 0부터 1까지 시작된다.
조건 : 명령을 반복하기 위해 조건을 체크합니다. 이조건을 만족해야 그 다음에 오는 명령을 실핼 할수 있다.
증가식 : 명령을 반복한 후 실행한다. 보통 카운터 변수를 1 증가시키는 용도로 사용된다.
ex) 
var i;
var sum = 0;
for(i=1; i < 6; i++){
    sum += i;
} 

while문
조건을 체크하고 True면 {} 안의 명령 실행 -> 조건이 false라면 명령은 한번도 실행하지 않을 수 있음
while(조건) {
    실행할명령
}
do ~ while문
일단 명령을 한번 실행한 후 조건 체크. True라면 ()안의 명령 실행,false라면{}을 빠져나옴
-> 조건이 false라도 명령은 최소한 한 번은 실행
do {
    실행할 명령
} while(조건)

let을 사용한 변수의 특징
- 블록변수 : 블록({}) 안에서만 사용할수 있다.
 -> 전역변수는 변수 이름과 초깃값만 할당하면 된다.
 -> 재할당은 가능하지만 재선언 할수 없다.
 ->호이스팅이 없다.
 ex)
function calcSum(n){
    sum = 0 -> 전역변수이다.
    for(let i = 1; i <n +1; i++){ -> let i = 1; 블록변수이다.
        sum += i 
    }
}

const를 사용한 변수의 특징
 -> 상수 : 변하지 않는 값을 선언할 때 사용
 -> 재선언,재할당할 수 없음

 ************** 자바스크립트 변수 이렇게 사용하자 ******************
 -> 전역변수는 최소한으로 사용
 -> var변수는 함수의 시작부분에 선언
 -> for문에서 카운터 변수는 var보다 let 변수로 선언
 -> ES6를 사용한다면 var 보다 let를 사용 하는것이 좋다.
